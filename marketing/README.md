# Trnscrbr Marketing

This folder contains launch assets for `trnscrbr.ai`.

## One-page Website

Open `site/index.html` directly, or run:

```bash
npm install
npm run site
```

Then visit `http://localhost:4173`.

## Remotion Intro Video

Preview the composition:

```bash
npm install
npm run video
```

Render the intro:

```bash
npm run render
```

The composition ID is `TrnscrbrIntro`.

## ImageGen Asset

`assets/trnscrbr-hero.png` was generated with the built-in ImageGen tool for the site hero and Remotion intro background.
