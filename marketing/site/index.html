Download and install

  1. Download the setup file. Use the button above to get the installer from GitHub Releases.
  2. Keep the file if your browser warns you. Trnscrbr is not yet code-signed or Microsoft-certified, so Edge or Chrome may offer choices such as Keep and Delete. Choose Keep only when the file came from this GitHub release link.
  3. Run the installer. Open the downloaded Trnscrbr-Setup-0.2.4-win-x64.exe file. If Windows SmartScreen appears, choose More info, then Run anyway.
  4. Start dictating. Launch Trnscrbr, complete setup, then use Ctrl + Win + Space to hold-to-talk from any normal text field.
