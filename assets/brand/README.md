# Trnscrbr Brand Assets

## Icon

- `trnscrbr-icon.svg` is the reusable vector icon for website, docs, and generated raster exports.
- `trnscrbr-icon-512.png` is a Discord/social-friendly raster export.
- It matches the app's idle blue/teal/purple voice-button state.

For Discord or app-store-style usage, export this SVG to PNG at `512x512` or higher.
